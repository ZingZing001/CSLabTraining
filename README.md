# CS Lab Training

This repository stores Jupyter Notebook files for each lab session.

## Structure

```
CSLabTraining/
├── lab01/
│   └── lab01.ipynb
├── lab02/
│   └── lab02.ipynb
├── lab03/
│   └── lab03.ipynb
└── ...
```

## Usage

1. **Clone** the repo on the lab computer:
   ```bash
   git clone https://github.com/ZingZing001/CSLabTraining.git
   ```
2. **Pull** the latest changes if already cloned:
   ```bash
   git pull
   ```
3. Work on your lab notebook in the corresponding `labXX/` folder.
4. **Commit & push** when done:
   ```bash
   git add .
   git commit -m "Complete labXX"
   git push
   ```
5. Submit the `.ipynb` file to the TA.
